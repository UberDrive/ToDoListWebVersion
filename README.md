This README file will not provide any helpful information for managing the project above; however, it will contain some of my thoughts as a junior software engineer.The amount of time and mental effort that AI is already helping to reduce is incomparable to what a software developer had to do and study before the release of advanced LLM versions such as Claude — models that are capable not just of helping with debugging, as we thought in the beginning, but of creating easy to medium difficulty projects if the instructions are given by someone who understands even a little bit of programming, or whatever the task may be.In my opinion, Claude-like models must not be artificially restricted or prevented from further progression.
